# 🎤 Voice Assistant (Python)

A simple voice-controlled assistant built using Python that can perform tasks like opening apps, playing songs, and responding to user commands.

## 🚀 Features

* Open YouTube, Google, Notepad
* Play songs using voice commands
* Tell current time
* Voice interaction using microphone

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3
* pywhatkit
* PyAudio

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```

## 📌 Example Commands

* "open youtube"
* "play shape of you"
* "what is the time"
* "goodbye"

## 📂 Project Structure

* main.py → main loop
* voice.py → speech input/output
* commands.py → actions
